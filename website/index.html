03 / FAQ

Questions, answered. Questions, réponses.

If something isn't here, the docs cover the rest. Si vous ne trouvez pas, la documentation couvre le reste.

Which platforms does Vocali support? Quelles plateformes sont supportées ?
Vocali ships native apps for macOS (Apple Silicon and Intel — universal binary) and Windows 10 / 11. The macOS app is a Swift menu-bar app; the Windows app is a Python tray app. Vocali existe en natif pour macOS (Apple Silicon et Intel — binaire universel) et Windows 10/11. L'app macOS est en Swift dans la barre de menu ; l'app Windows est en Python dans la barre des tâches.
Why Groq by default? Pourquoi Groq par défaut ?
Groq's transcription and inference are fast enough (under a second end-to-end) that dictation feels instant. Local models still hover around 5–10 seconds, which breaks the flow you came here for. You can swap providers any time. Transcription et inférence chez Groq tournent sous la seconde de bout en bout, ce qui rend la dictée instantanée. Les modèles locaux restent autour de 5 à 10 secondes, ce qui casse le flow. Vous pouvez changer de fournisseur à tout moment.
Does Vocali send my voice anywhere? Vocali envoie-t-il ma voix quelque part ?
Only to the API endpoint you configure. There is no Vocali server in the loop — no telemetry, no analytics on what you say, no cloud sync. The default endpoint is Groq; point it at OpenAI, a self-hosted Whisper, anything OpenAI-compatible. Uniquement à l'endpoint que vous configurez. Aucun serveur Vocali dans la boucle — pas de télémétrie, pas d'analytique sur ce que vous dites, pas de sync cloud. L'endpoint par défaut est Groq ; pointez vers OpenAI, un Whisper auto-hébergé, ou tout endpoint compatible.
What does it cost? Combien ça coûte ?
Vocali itself is free. You pay your API provider directly for transcription and cleanup tokens — typically pennies per hour of dictation on Groq's free tier. Vocali est gratuit. Vous payez directement votre fournisseur d'API pour les tokens de transcription et de nettoyage — généralement quelques centimes par heure de dictée sur le tier gratuit Groq.
Will my custom shortcuts conflict with system ones? Mes raccourcis vont-ils entrer en conflit ?
Vocali warns you when a binding clashes with a known system or app shortcut, and lets you scope shortcuts per application. Hold-to-talk on Fn (Mac) or Right Alt (Windows) is the default because it's almost always free. Vocali vous prévient si un raccourci entre en conflit avec un raccourci système ou applicatif connu, et permet de définir des raccourcis par application. Hold-to-talk sur Fn (Mac) ou Alt droit (Windows) est par défaut parce que c'est presque toujours libre.
Can I use Vocali offline? Vocali fonctionne-t-il hors ligne ?
Not yet. The cleanup pass needs a model and the realtime transcription needs network. A local-model option is on the roadmap once on-device latency is acceptable. Pas encore. Le nettoyage et la transcription temps réel nécessitent un modèle et donc du réseau. Une option modèle local est dans la roadmap dès que la latence sur appareil sera acceptable.
